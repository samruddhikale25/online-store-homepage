## 📌 Project Overview

This project is developed as part of an internship task to create a dynamic and responsive e-commerce homepage.  
It allows users to browse products, filter them based on categories, sort them by price, and search products easily.

---

## ✨ Features

- 🧾 Product listing with attractive UI cards  
- 📂 Category-based filtering  
- 💰 Price sorting (Low to High / High to Low)  
- 🔍 Real-time search functionality  
- 🛒 Add to Cart (basic functionality)  
- 📱 Fully responsive design  
- 🎨 Modern UI using Tailwind CSS  

---

## 🛠️ Technologies Used

- HTML5  
- Tailwind CSS  
- JavaScript (Vanilla JS)
